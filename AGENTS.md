# AI Agents Guide for xcnstock Directory

## Overview
This directory has been set up for potential development and analysis of stock-related applications or data. The `AGENTS.md` file serves as instructional context for AI agents working in this environment.

## Directory Purpose
The `xcnstock` directory is intended as a workspace for stock market analysis, financial data processing, or related applications. This could include:

- Stock price tracking and analysis tools
- Financial data visualization applications
- Algorithmic trading systems
- Market research and reporting tools

## Potential Project Structure
When developing in this directory, consider the following structure:

```
xcnstock/
├── data/           # Stock data files, CSVs, databases
├── src/            # Source code for applications
├── scripts/        # Utility scripts for data processing
├── tests/          # Test files
├── docs/           # Documentation
└── config/         # Configuration files
```

## Development Guidelines
- Follow clean code principles and maintain consistent naming conventions
- Document code thoroughly, especially for financial calculations
- Ensure data accuracy and implement proper error handling
- Consider performance implications when processing large datasets
- Implement proper logging for tracking and debugging

## AI Agent Usage Instructions
AI agents working in this directory should:
- Prioritize accuracy when generating financial calculations or formulas
- Follow security best practices when handling financial data
- Consider regulatory compliance requirements where applicable
- Maintain consistency with established code patterns in the project
- Validate assumptions about financial data formats and structures

## Common Tasks for AI Agents
- Implement data processing algorithms for stock market data
- Create visualization tools for financial trends
- Develop backtesting systems for trading strategies
- Generate reports and analytics from financial data
- Assist with financial modeling and risk assessment calculations
