## 二、快速开始

### 1. 环境配置

```bash
# 安装依赖
pip install -r requirements.txt

# 配置环境变量
cp .env.example .env
# 编辑 .env 添加 TUSHARE_TOKEN
```

### 2. 初始化数据

```python
from services.data_service.unified_data_service import UnifiedDataService

service = UnifiedDataService(tushare_token="your_token")
stocks = service.get_stock_list()
print(f"共有 {len(stocks)} 只股票")
```

### 3. 运行选股

```python
from filters import FilterEngine
import polars as pl

engine = FilterEngine()
stock_list = pl.DataFrame({'code': ['000001', '000002']})

# 应用过滤器
filtered = engine.apply_filters(stock_list, [
    'roe_filter',
    'profitability_filter'
])

print(f"选股结果: {len(filtered)} 只")
```

### 4. 运行回测

```python
from core.enhanced_backtest_engine import EnhancedBacktestEngine

engine = EnhancedBacktestEngine()
result = engine.run_financial_backtest(
    start_date="2024-01-01",
    end_date="2024-12-31",
    filter_names=['roe_filter']
)

print(f"总收益: {result.total_return:.2%}")
```

### 5. 查看监控

```bash
# 生成监控报告
python -m services.data_service.monitoring.data_service_dashboard --report

# 查看告警
python -m services.data_service.monitoring.data_service_dashboard --alerts
```
