# 每日自动化工作流程设计

## 概述

XCNStock 每日自动化运行方案，实现无人值守的全流程股票分析与推送。

## 设计目标

- **全自动化运行** - 无需人工干预
- **定时触发** - 关键时间点自动执行
- **多渠道推送** - 邮件 + 微信 + 钉钉 + Kafka

---

│                                                                      │

## 二、任务详情

### 盘前/盘中任务

| 时间 | 任务 | 输出 | 推送级别 | 渠道 |
|------|------|------|---------|------|
| 08:00 | 报告检查更新 | 验证报告 | - | - |
| 08:30 | 次日操作报告 | 选股+打板+热点 | C级 | 邮件+微信+钉钉+Kafka |
| 09:25 | 集合竞价异动 | 高开>3%股票 | A级 | 邮件+微信+钉钉+Kafka |
| 09:30 | 开盘处理 | 一字涨停标记 | - | - |
| 11:30 | 午盘总结 | 涨停板汇总 | B级 | 邮件+微信 |
| 14:30 | 尾盘监控 | 封板/开板预警 | S级 | 邮件+微信+钉钉+Kafka |

### 盘后任务

| 时间 | 任务 | 输出 | 说明 |
|------|------|------|------|
| 15:30 | 数据采集 | Parquet文件 | K线、行情、涨停板 |
| 16:00 | 数据验证 | 审计报告 | 完整性、新鲜度检查 |
| 16:30 | 当日复盘 | 复盘报告 | 涨停/热点/资金分析 |
| 17:00 | 次日选股 | 选股报告 | S/A级推荐、打板名单 |
| 18:00 | 报告生成 | JSON/HTML | 待次日推送 |

## 三、一字涨停处理

### 判断规则

| 情况 | 判断条件 | 处理方式 |
|------|---------|---------|
| 一字涨停 | 开盘价=涨停价 且 成交量极低 | 标记"一字板"，从打板名单剔除 |
| 高开涨停 | 开盘涨幅>9% 但非一字 | 标记"高开涨停"，降级处理 |
| 正常开盘 | 开盘涨幅<9% | 正常监控 |

### 报告展示

```
【打板名单】
  ✓ 正常涨停股 (可操作)
    002235 安妮股份 +9.99%
    
  ⊘ 一字涨停股 (已剔除)
    300042 朗科科技 +20.00% [一字板]
```

## 四、推送配置

### 推送渠道

| 渠道 | 配置 | 用途 |
|------|------|------|
| 邮件 | SMTP | 所有信号都发送 |
| 微信 | Server酱 | 实时通知 |
| 钉钉 | Webhook机器人 | 重要信号+@提醒 |
| Kafka | 49.233.10.199:9092 | 股票推荐/打板数据 |

### 推送分级

| 信号级别 | 邮件 | 微信 | 钉钉 | Kafka |
|---------|------|------|------|-------|
| S级(紧急) | 即时 | 即时 | 即时+@ | 即时 |
| A级(重要) | 即时 | 即时 | 即时 | 即时 |
| B级(普通) | 即时 | 汇总 | - | - |
| C级(日常) | 即时 | 即时 | 即时 | 即时 |

## 六、报告内容

### 次日操作报告

```
1. S级推荐
   - 多头排列+量价齐升
   - MACD金叉+趋势确认
   
2. A级关注
   - 高评分潜力股
   
3. 打板名单
   - 今日涨停+多头排列(连板潜力)
   - 涨停+高评分(质量优选)
   - 已剔除一字涨停
   
4. 热点追踪
   - 持续热点板块
   - 新兴热点板块
   
5. 重大机会
   - S级涨停股
   - 机构/北向大买
```

### 当日复盘报告

```
1. 大盘分析
   - 指数涨跌、成交量、趋势判断
   
2. 涨停板分析
   - 涨停数量、板块分布、连板统计
   - 封板强度排名、开板股票
   
3. 热点板块
   - 涨幅前5板块、资金流入板块
   - 板块龙头股、跟风股
   
4. 资金动向
   - 北向资金、主力资金
   - 大单异动、龙虎榜
```
