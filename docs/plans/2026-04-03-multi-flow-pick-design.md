# 多流程选股对比系统设计

## 1. 概述

### 1.1 目标
同时运行三种选股流程，对比不同策略的持仓表现，找出最优选股流程。

### 1.2 三种流程

| 流程 | 名称 | 特点 | 风险 |
|------|------|------|------|
| A | 过滤优先 | 过滤→因子→共振→选股 | 保守 |
| B | 因子优先 | 全量因子→综合评分→选股 | 均衡 |
| C | 信号优先 | 纯共振信号→验证→选股 | 激进 |

---

## 2. 流程详细设计

### 流程A: 过滤优先 (ConservativeFilter)

```
1. 读取K线数据
2. 过滤: 剔除涨跌停/停牌/低流动性/科创创业
3. 计算基础因子 (MA、MACD、KDJ)
4. 计算主力共振信号
5. 综合评分: 因子40% + 共振信号60%
6. 选股: 取评分前30只
```

### 流程B: 因子优先 (BalancedFactor)

```
1. 读取K线数据 (不预过滤)
2. 计算全量因子 (技术15个 + 量价8个 + 市场5个)
3. 综合评分: 多因子加权平均
4. 选股: 取评分前30只
```

### 流程C: 信号优先 (AggressiveSignal)

```
1. 读取K线数据
2. 计算主力共振信号 (S+/A/B级)
3. 过滤: 剔除不满足买入条件的
4. 选股: S+全取 + A取前50% + B取前20%
```

## 5. 对比分析

### 5.1 追踪指标

| 指标 | 说明 |
|------|------|
| 次日收益率 | 选股后第一天收盘涨幅 |
| 3日收益率 | 选股后第3天收盘涨幅 |
| 5日收益率 | 选股后第5天收盘涨幅 |
| 最大回撤 | 持仓期间最大跌幅 |
| 胜率 | 正收益次数/总次数 |
| 平均收益 | 所有持仓的平均收益 |

### 5.2 输出报告

- 每日选股报告 (JSON)
- 周度对比分析 (Markdown)
- 月度汇总报告 (Markdown)

## 7. 执行时机

- 每日 18:00 执行
- 次日 10:00 汇总前一交易日表现
- 每周一生成上周对比报告
