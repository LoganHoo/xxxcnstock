# 主力痕迹共振选股策略设计

**日期**: 2026-04-03
**状态**: 已批准

---

## 1. 概述

识别A股市场主力资金的四大真实痕迹，当四项信号共振时，表明主力已完成布局，主升浪启动信号明确。

## 2. 四大信号定义

| 信号ID | 名称 | 条件 | 技术含义 |
|--------|------|------|----------|
| S1 | 高质量涨停 | 20日内有涨停 + 量能≥前日1.5倍 + 之后10日横盘不跌 | 主力试盘+洗盘完成 |
| S2 | 未回补跳空 | 3日内存在向上跳空缺口，且至今未回补 | 强势突破，主力成本区上方 |
| S3 | 标准连阳 | 连续3日收盘价创新高 + 每日最低价≥前日收盘价 | 连续吸筹，不给回调机会 |
| S4 | 关键位放量 | 放量突破日均量2倍 + 站稳放量区3日 | 主力资金确认进入 |

## 3. 信号检测算法

### S1: 高质量涨停检测
```
条件1: 20日内任意一天涨停 (close >= prev_close * 1.095)
条件2: 涨停当日量能 >= 前日量能 * 1.5
条件3: 涨停后10日内:
       - 最低价 > 涨停日收盘价 * 0.97 (不跌破涨停价3%)
       - 无跌停 (close >= prev_close * 0.9)
```

### S2: 未回补跳空缺口检测
```
条件: 3日内存在向上跳空缺口
      gap = today_low > yesterday_high
      且缺口至今未被回补:
      - 从缺口日后第3天开始检查
      - 如果最低价仍 > 缺口日最高价，则未回补
```

### S3: 标准连阳检测
```
条件: 连续3日满足:
      - 收盘价 > 前日收盘价 (每日创新高)
      - 最低价 >= 前日收盘价 (不破前日最低)
      即: close[i] > close[i-1] AND low[i] >= close[i-1]
```

### S4: 关键位放量检测
```
条件: 今日量能 >= 20日平均量能 * 2
      且最近3日内任意一天放量时:
      - 收盘价 > 放量日最低价
      - 站稳放量区域
```

## 4. 评分规则

| 信号计数 | 级别 | 说明 |
|----------|------|------|
| 4 | S+ | 主升浪启动信号明确 |
| 3 | A | 重点关注 |
| 2 | B | 候选观察 |
| 1 | C | 弱候选 |
