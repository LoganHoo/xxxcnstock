## 选股过滤

### 3.1 使用单个过滤器

```python
from filters import ROEFilter, DragonTigerFilter, PerformanceForecastFilter
import polars as pl

# 准备股票列表
stock_list = pl.DataFrame({
    'code': ['000001', '000002', '600000', '000858', '600519'],
    'name': ['平安银行', '万科A', '浦发银行', '五粮液', '贵州茅台'],
})

# ROE过滤 (ROE >= 15%)
roe_filter = ROEFilter(params={'min_roe': 15.0})
filtered = roe_filter.filter(stock_list)
print(f"ROE >= 15% 的股票: {len(filtered)} 只")

# 龙虎榜过滤
# dragon_tiger_filter = DragonTigerFilter(params={'min_institution_net': 500})
# filtered = dragon_tiger_filter.filter(stock_list)

# 业绩预告过滤
# forecast_filter = PerformanceForecastFilter(params={'forecast_types': ['预增', '扭亏']})
# filtered = forecast_filter.filter(stock_list)
```

### 3.2 使用过滤器引擎

```python
from filters import FilterEngine

# 创建过滤器引擎
engine = FilterEngine()

# 列出所有可用过滤器
filters = engine.list_filters()
for f in filters:
    print(f"{f['name']}: {f['description']} (启用: {f['enabled']})")

# 应用多个过滤器
filtered = engine.apply_filters(stock_list, [
    'roe_filter',
    'profitability_filter',
    'growth_filter'
])
print(f"过滤后: {len(filtered)} 只")
```

## 数据更新

### 4.1 手动更新

```python
from services.data_service.tasks import DailyUpdateTask

# 创建更新任务
task = DailyUpdateTask()

# 执行每日更新
results = task.run_daily_update()
print(f"更新完成: {results}")

# 执行财务数据季度更新
# financial_results = task.run_financial_update()
```

### 4.2 增量更新

```python
from services.data_service.tasks import IncrementalUpdateTask

# 创建增量更新任务
task = IncrementalUpdateTask(max_workers=4)

# 执行增量更新
results = task.run_incremental_financial_update(
    stock_codes=['000001', '000002', '600000'],
    years=3,
    resume=True  # 支持断点续传
)

print(f"成功: {results['success_count']}, 失败: {results['fail_count']}")
print(f"新增记录: {results['added_records']}")
```

### 4.3 数据预热

```python
from services.data_service.tasks import DataPreheatingTask

# 创建预热任务
task = DataPreheatingTask(max_workers=4)

# 检查是否需要预热
if task.is_preheat_needed(max_age_minutes=60):
    # 执行预热
    result = task.run_preheating()
    print(f"预热完成: {result['success']} 只成功, 耗时 {result['elapsed_time']:.2f}秒")
else:
    print("数据已预热，无需再次预热")
```

## 回测

### 5.1 财务数据回测

```python
from core.enhanced_backtest_engine import EnhancedBacktestEngine, RebalanceFreq

# 创建回测引擎
engine = EnhancedBacktestEngine()

# 运行财务数据回测
result = engine.run_financial_backtest(
    start_date="2024-01-01",
    end_date="2024-12-31",
    initial_capital=1000000,
    position_size=10,
    rebalance_freq=RebalanceFreq.QUARTERLY,
    filter_names=['roe_filter', 'profitability_filter']
)

print(f"总收益: {result.total_return:.2%}")
print(f"年化收益: {result.annualized_return:.2%}")
print(f"最大回撤: {result.max_drawdown:.2%}")
print(f"胜率: {result.win_rate:.2%}")
```

### 5.2 综合策略回测

```python
# 运行综合策略回测
result = engine.run_composite_backtest(
    start_date="2024-01-01",
    end_date="2024-12-31",
    financial_filters=['roe_filter', 'profitability_filter'],
    market_behavior_filters=['dragon_tiger_filter'],
    announcement_filters=['performance_forecast_filter']
)

# 保存回测结果
engine.save_result(result, filename="my_backtest.json")
```

## 常见问题

### Q1: 如何添加新的数据源？

A: 实现 `FinancialDataSource` 接口：

```python
from services.data_service.fetchers.financial.base import FinancialDataSource

class MyDataSource(FinancialDataSource):
    def fetch_balance_sheet(self, code: str, years: int = 3) -> pd.DataFrame:
        # 实现数据获取逻辑
        pass
```

### Q2: 如何自定义过滤器？

A: 继承过滤器基类并注册：

```python
from filters import FinancialFilterBase, register_filter

@register_filter("my_filter")
class MyFilter(FinancialFilterBase):
    def filter(self, stock_list: pl.DataFrame) -> pl.DataFrame:
        # 实现过滤逻辑
        return filtered_stocks
```

### Q3: 数据更新失败怎么办？

A: 检查以下几点：
1. API Token 是否有效
2. 网络连接是否正常
3. 查看日志文件 `logs/system/`
4. 使用增量更新任务的断点续传功能

### Q4: 如何提高查询性能？

A: 
1. 启用内存缓存 (默认启用)
2. 使用数据预热功能
3. 批量查询代替单只查询
4. 使用优化的存储管理器

### Q5: 如何配置定时任务？

A: 编辑 `config/cron_tasks.yaml`：

```yaml
tasks:
  daily_update:
    schedule: "0 17 * * *"  # 每天17:00
    command: "python -m services.data_service.tasks.daily_update_task"
  
  data_preheating:
    schedule: "30 8 * * *"  # 每天8:30
    command: "python -m services.data_service.tasks.data_preheat_task"
```

## 附录

### A. 配置文件参考

**财务过滤器配置** (`config/filters/financial_filter.yaml`):
```yaml
filters:
  roe_filter:
    enabled: true
    min_roe: 15.0
    max_roe: 100.0
```

**市场行为过滤器配置** (`config/filters/market_behavior_filter.yaml`):
```yaml
filters:
  dragon_tiger_filter:
    enabled: true
    min_institution_net: 500
```

### B. 环境变量

| 变量名 | 说明 | 必需 |
|--------|------|------|
| TUSHARE_TOKEN | Tushare API Token | 是 |
| DATA_DIR | 数据存储目录 | 否 |
| LOG_LEVEL | 日志级别 | 否 |

### C. 日志文件

| 日志文件 | 说明 |
|---------|------|
| `logs/system/financial_storage.log` | 财务数据存储日志 |
| `logs/system/data_quality_monitor.log` | 数据质量监控日志 |
| `logs/system/incremental_update.log` | 增量更新日志 |
| `logs/system/data_preheating.log` | 数据预热日志 |
