## 一、数据采集业务流

### 1.1 财务数据采集流程

```
┌─────────────────────────────────────────────────────────────────────────┐
│                        财务数据采集业务流程                              │
└─────────────────────────────────────────────────────────────────────────┘

触发条件
    │
    ├── 定时触发: 季度报告期后 (4月、8月、10月)
    ├── 手动触发: 用户执行更新命令
    └── 事件触发: 检测到新报告发布
    │
    ▼
┌─────────────┐
│  获取股票列表 │ ← 从股票基础数据获取全部A股代码
└──────┬──────┘
       │
       ▼
┌─────────────┐     ┌─────────────┐     ┌─────────────┐
│  资产负债表  │ ←→ │  利润表     │ ←→ │  现金流量表  │
│  数据获取   │     │  数据获取   │     │  数据获取   │
└──────┬──────┘     └──────┬──────┘     └──────┬──────┘
       │                   │                   │
       └───────────────────┼───────────────────┘
                           │
                           ▼
┌─────────────────────────────────────────────────────────┐
│  数据验证                                                │
│  ├── 会计恒等式验证: 资产 = 负债 + 所有者权益            │
│  ├── 数据完整性验证: 必填字段检查                        │
│  ├── 数据合理性验证: 数值范围检查                        │
│  └── 时间连续性验证: 报告期连续性                        │
└─────────────────────────────────────────────────────────┘
                           │
                           ▼
┌─────────────────────────────────────────────────────────┐
│  财务指标计算                                            │
│  ├── 盈利能力: ROE, ROA, 毛利率, 净利率                  │
│  ├── 偿债能力: 资产负债率, 流动比率, 速动比率            │
│  ├── 成长能力: 营收增长率, 净利润增长率                  │
│  └── 现金流: 经营现金流/净利润, 自由现金流               │
└─────────────────────────────────────────────────────────┘
                           │
                           ▼
┌─────────────────────────────────────────────────────────┐
│  数据存储                                                │
│  ├── 资产负债表 → data/financial/balance_sheet/         │
│  ├── 利润表 → data/financial/income_statement/          │
│  ├── 现金流量表 → data/financial/cash_flow/             │
│  └── 财务指标 → data/financial/indicators/              │
│  格式: Parquet, 压缩: Zstd                               │
└─────────────────────────────────────────────────────────┘
                           │
                           ▼
┌─────────────────────────────────────────────────────────┐
│  质量报告生成                                            │
│  ├── 数据质量评分 (0-100)                               │
│  ├── 问题记录与分类                                      │
│  └── 存储至 data/quality_reports/                       │
└─────────────────────────────────────────────────────────┘
```

**实现状态**: ✅ 已实现
- 数据源: Baostock (主)

## 二、选股策略业务流

### 2.1 标准选股流程

```
┌─────────────────────────────────────────────────────────────────────────┐
│                          标准选股业务流程                                │
└─────────────────────────────────────────────────────────────────────────┘

用户请求
    │
    ├── 输入参数:
    │   ├── 选股范围: 全市场 / 指定板块 / 自选股
    │   ├── 选股策略: 财务指标 / 市场行为 / 公告事件 / 综合
    │   └── 输出数量: Top N (默认50)
    │
    ▼
┌─────────────────────────────────────────────────────────┐
│  步骤1: 股票池准备                                       │
│  ├── 获取股票基础列表                                    │
│  ├── 过滤停牌股票                                        │
│  ├── 过滤ST股票                                          │
│  └── 过滤退市股票                                        │
└─────────────────────────────────────────────────────────┘
                           │
                           ▼
┌─────────────────────────────────────────────────────────┐
│  步骤2: 数据加载                                         │
│  ├── 加载财务数据 (缓存优先)                             │
│  ├── 加载市场行为数据                                    │
│  └── 加载公告数据                                        │
└─────────────────────────────────────────────────────────┘
                           │
                           ▼
┌─────────────────────────────────────────────────────────┐
│  步骤3: 过滤器链执行                                     │
│                                                         │
│  财务指标过滤 (可选)                                     │
│  ├── ROE >= 15%                                         │
│  ├── 毛利率 >= 30%                                       │
│  ├── 营收增长率 >= 15%                                   │
│  └── 资产负债率 <= 60%                                   │
│                                                         │
│  市场行为过滤 (可选)                                     │
│  ├── 机构净买入 >= 500万                                 │
│  ├── 主力净流入 >= 1000万                                │
│  └── 北向资金持股                                        │
│                                                         │
│  公告事件过滤 (可选)                                     │
│  ├── 业绩预告: 预增/扭亏                                 │
│  ├── 重大事项: 并购重组                                  │
│  └── 股权变动: 增持                                      │
└─────────────────────────────────────────────────────────┘
                           │
                           ▼
┌─────────────────────────────────────────────────────────┐
│  步骤4: 综合评分与排序                                   │
│  ├── 财务评分 (权重40%)                                  │
│  ├── 市场行为评分 (权重30%)                              │
│  ├── 公告事件评分 (权重20%)                              │
│  └── 其他因子 (权重10%)                                  │
└─────────────────────────────────────────────────────────┘
                           │
                           ▼
┌─────────────────────────────────────────────────────────┐
│  步骤5: 结果输出                                         │
│  ├── 股票代码、名称                                      │
│  ├── 关键财务指标                                        │
│  ├── 市场行为指标                                        │
│  ├── 相关公告                                            │
│  └── 综合评分与排名                                      │
└─────────────────────────────────────────────────────────┘
```

## 三、回测验证业务流

### 3.1 回测流程

```
┌─────────────────────────────────────────────────────────────────────────┐
│                          回测验证业务流程                                │
└─────────────────────────────────────────────────────────────────────────┘

用户请求
    │
    ├── 输入参数:
    │   ├── 回测区间: 开始日期 - 结束日期
    │   ├── 初始资金: 默认100万
    │   ├── 选股策略: 过滤器组合
    │   ├── 调仓频率: 日度/周度/月度/季度
    │   └── 仓位管理: 等权重 / 市值加权
    │
    ▼
┌─────────────────────────────────────────────────────────┐
│  步骤1: 历史数据准备                                     │
│  ├── 获取回测区间内所有交易日                            │
│  ├── 获取股票历史价格数据                                │
│  ├── 获取财务历史数据                                    │
│  └── 获取市场行为历史数据                                │
└─────────────────────────────────────────────────────────┘
                           │
                           ▼
┌─────────────────────────────────────────────────────────┐
│  步骤2: 回测引擎初始化                                   │
│  ├── 设置初始资金                                        │
│  ├── 设置手续费率 (默认0.03%)                            │
│  ├── 设置滑点 (默认0.01%)                                │
│  └── 设置调仓规则                                        │
└─────────────────────────────────────────────────────────┘
                           │
                           ▼
┌─────────────────────────────────────────────────────────┐
│  步骤3: 逐日模拟交易                                     │
│                                                         │
│  对每个交易日:                                           │
│  ├── 检查是否调仓日                                      │
│  │   ├── 是: 执行选股策略                                │
│  │   │       ├── 获取当前符合条件的股票                  │
│  │   │       ├── 计算目标持仓                            │
│  │   │       ├── 生成调仓指令 (买入/卖出)                │
│  │   │       └── 执行交易 (考虑手续费和滑点)             │
│  │   └── 否: 保持当前持仓                                │
│  │                                                         │
│  ├── 更新持仓市值 (使用当日收盘价)                       │
│  └── 记录每日资产净值                                    │
│                                                         │
└─────────────────────────────────────────────────────────┘
                           │
                           ▼
┌─────────────────────────────────────────────────────────┐
│  步骤4: 绩效计算                                         │
│  ├── 总收益率 = (期末资产 - 期初资产) / 期初资产         │
│  ├── 年化收益率                                          │
│  ├── 最大回撤                                            │
│  ├── 夏普比率                                            │
│  ├── 胜率 (盈利交易次数 / 总交易次数)                    │
│  ├── 盈亏比 (平均盈利 / 平均亏损)                        │
│  └── 阿尔法/贝塔 (相对基准指数)                          │
└─────────────────────────────────────────────────────────┘
                           │
                           ▼
┌─────────────────────────────────────────────────────────┐
│  步骤5: 生成回测报告                                     │
│  ├── 收益曲线图                                          │
│  ├── 回撤曲线图                                          │
│  ├── 月度收益热力图                                      │
│  ├── 交易记录明细                                        │
│  ├── 持仓分析                                            │
│  └── 风险指标汇总                                        │
└─────────────────────────────────────────────────────────┘
```

**实现状态**: ✅ 已实现
- 实现文件: `core/historical_backtest_engine.py`
- 支持调仓频率: 日度/周度/月度
- 支持历史数据切片

## 六、异常处理业务流

### 6.1 数据获取异常

```
异常: 数据获取失败
    │
    ├── 自动重试 (最多3次)
    │   ├── 成功 → 继续流程
    │   └── 失败 → 切换备用数据源
    │
    ├── 备用数据源获取
    │   ├── 成功 → 继续流程
    │   └── 失败 → 记录异常
    │
    └── 人工介入
        ├── 检查网络连接
        ├── 检查API Token
        ├── 联系数据源支持
        └── 手动补录数据
```

### 6.2 数据质量异常

```
异常: 数据质量评分低于阈值
    │
    ├── 自动处理
    │   ├── 标记问题数据
    │   ├── 发送告警通知
    │   └── 尝试自动修复 (如可能)
    │
    └── 人工介入
        ├── 分析问题原因
        ├── 修复数据问题
        ├── 更新验证规则
        └── 重新验证数据
```
