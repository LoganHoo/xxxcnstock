## 二、数据工作流程

### 2.1 数据采集流程

```
┌─────────────┐     ┌─────────────┐     ┌─────────────┐     ┌─────────────┐
│  数据源选择  │ →  │  数据获取   │ →  │  数据验证   │ →  │  数据存储   │
│             │     │             │     │             │     │             │
│ Tushare Pro │     │ 多线程获取  │     │ 格式验证    │     │ Parquet存储 │
│ AKShare(备) │     │ 失败重试    │     │ 完整性检查  │     │ 索引建立    │
└─────────────┘     └─────────────┘     └─────────────┘     └─────────────┘
```

### 2.2 数据更新流程

```
定时触发 → 增量检测 → 差异对比 → 数据获取 → 验证 → 合并 → 存储 → 缓存更新 → 质量检查
   │         │          │          │        │      │      │        │          │
   ▼         ▼          ▼          ▼        ▼      ▼      ▼        ▼          ▼
 Cron    对比日期    计算差异   API调用   格式   合并   写入   刷新缓存   生成报告
```

### 2.3 数据质量检查流程

```
┌─────────────────┐
│   数据质量检查   │
├─────────────────┤
│ 1. 新鲜度检查    │ → 最新数据日期是否在阈值内
│ 2. 完整性检查    │ → 必填字段是否完整
│ 3. 一致性检查    │ → 会计恒等式是否成立
│ 4. 合理性检查    │ → 数值是否在合理范围
│ 5. 连续性检查    │ → 时间序列是否连续
└─────────────────┘
         │
         ▼
   生成质量报告
   - 质量评分 (0-100)
   - 问题列表
   - 改进建议
```

## 三、选股工作流程

### 3.1 标准选股流程

```
┌─────────────┐     ┌─────────────┐     ┌─────────────┐     ┌─────────────┐
│  股票池准备  │ →  │  过滤器链   │ →  │  评分排序   │ →  │  输出结果   │
│             │     │             │     │             │     │             │
│ 全市场股票  │     │ 财务过滤器  │     │ 综合评分    │     │ Top N股票   │
│ 或指定列表  │     │ 市场过滤器  │     │ 多因子排序  │     │ 详细指标    │
│             │     │ 公告过滤器  │     │             │     │ 买入建议    │
└─────────────┘     └─────────────┘     └─────────────┘     └─────────────┘
```

### 3.2 过滤器链配置

```yaml
# 示例: 财务+主力资金综合策略
选股流程:
  步骤1_财务筛选:
    - roe_filter (ROE >= 15%)
    - profitability_filter (毛利率 >= 30%)
    - growth_filter (营收增长 >= 15%)
    
  步骤2_主力资金筛选:
    - main_force_filter (机构净买入 >= 500万)
    
  步骤3_公告事件筛选:
    - performance_forecast_filter (预增/扭亏)
    
  步骤4_评分排序:
    - financial_composite_filter (综合评分排序)
```

### 3.3 回测验证流程

```
策略定义 → 历史数据加载 → 信号生成 → 模拟交易 → 绩效计算 → 报告生成
    │           │            │          │          │          │
    ▼           ▼            ▼          ▼          ▼          ▼
 过滤器链    财务数据     买入信号    仓位管理    收益率     回测报告
 参数配置    市场数据     卖出信号    调仓执行    最大回撤    图表分析
            公告数据
```

| 时间 | 任务 | 说明 |

| 8:30 | 数据预热 | 预热热点股票数据 |

```
告警触发 → 故障定位 → 影响评估 → 应急处理 → 根因分析 → 修复验证 → 复盘总结

    ▼          ▼          ▼          ▼          ▼          ▼          ▼
  邮件/短信   日志分析    影响范围   降级/切换   代码审查   测试验证   文档更新
  监控面板    链路追踪    数据损失   重启服务    配置检查   回归测试   流程改进
```

## 五、数据服务调用流程

### 5.1 统一服务入口

```python
# 1. 初始化服务
from services.data_service.unified_data_service import UnifiedDataService
service = UnifiedDataService(tushare_token="your_token")

# 2. 获取数据
financial_data = service.get_all_financial_data("000001")
market_data = service.get_dragon_tiger("20240101")
announcements = service.get_announcements("000001", days=7)

# 3. 选股过滤
from filters import FilterEngine
engine = FilterEngine()
filtered = engine.apply_filters(stock_list, ['roe_filter', 'main_force_filter'])

# 4. 回测验证
from core.enhanced_backtest_engine import EnhancedBacktestEngine
backtest_engine = EnhancedBacktestEngine()
result = backtest_engine.run_composite_backtest(...)

# 5. 监控检查
from services.data_service.monitoring import DataServiceDashboard
dashboard = DataServiceDashboard()
dashboard.print_report()
```
